# Proyecto Landing Page - Kinesiología

## 1. Estructura del Proyecto

El proyecto sigue una estructura limpia y ordenada para facilitar su mantenimiento y escalabilidad.

- **index.html**: Archivo principal. Contiene la estructura semántica y el contenido de la landing page.
- **css/**: Carpeta para hojas de estilo.
  - `styles.css`: Contiene todos los estilos, variables y utilidades responsivas.
- **js/**: Carpeta para scripts.
  - `script.js`: Maneja interacciones mínimas como el scroll suave y animaciones de entrada.
- **assets/images/**: Carpeta reservada para las imágenes del proyecto (actualmente vacía, lista para agregar fotos reales).

## 5. Lógica de Conversión

La página ha sido diseñada siguiendo principios psicológicos y de UX para maximizar la conversión (agendar una cita).

1.  **Impacto Inmediato (Hero)**: 
    -   Titular centrado en el beneficio ("Vive sin dolor"), no en la característica.
    -   Imagen aspiracional (se recomienda usar una foto de alta calidad de un paciente feliz o tratamiento).
    -   CTA (Call to Action) principal con alto contraste y efecto "pulse" para captar la atención.

2.  **Micro-Recompensas (Beneficios)**:
    -   Inmediatamente después del Hero, se muestran 3 beneficios claros para confirmar al usuario que está en el lugar correcto.
    -   Iconos visuales para lectura rápida (< 3 segundos).

3.  **Descubrimiento y Autoridad (Servicios y Confianza)**:
    -   Listado claro de servicios para filtrar al público objetivo.
    -   Sección de Testimonios (Prueba Social) para reducir la incertidumbre y generar confianza.

4.  **Facilidad Cognitiva**:
    -   Diseño limpio con mucho espacio en blanco.
    -   Tipografía legible y jerarquía visual clara.
    -   Colores profesionales (Teal/Azul) que transmiten calma y salud, con acentos en Naranja para los botones de acción.

5.  **CTA Omnipresente**:
    -   Botón de "Agendar" fijo en el header (visible al hacer scroll).
    -   Botones de WhatsApp pre-configurados con mensajes personalizados para reducir la fricción del usuario al iniciar la conversación.
    -   CTA Final grande y directo para recoger a los usuarios que leyeron toda la página.

## Instrucciones de Uso

1.  Abra `index.html` en su navegador web.
2.  Reemplace los textos de marcador de posición (como "Clínica X") con la información real final.
3.  Agregue imágenes reales a la carpeta `assets/images` y enlácelas en el HTML si desea agregar fotos (actualmente el diseño es limpio y tipográfico, pero soporta imágenes).
